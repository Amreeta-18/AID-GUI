(What is a persona?) Personas represent your users. The GenderMag personas are Abi, Pat, and Tim. Abi's cognitive style consists of a set of facet values at one end of the cognitive style spectrum. Tim is at the other end. Pat is somewhere in between. Most people are a mixture of Abi and Tim---a mixture of facet values. In addition, an individual's facet values can change with time and circumstances. Abi provides the strongest inclusion focus and is the only persona currently being used for this automated evaluation.

A subgoal is what you as the instructor hope the student is thinking of doing next. Subgoals usually start with a verb, like “Find” something, “Create” something or “Add” something.
They’re different from Actions; these involve clicking on or tapping something. It’s important to make that distinction.

Follow the instructions below to save a HTML file:
In Chrome:

  • Navigate to the web page, right-click on the page and select Save as.
  • elect or create a new folder to save the file, images, and associated items from the web page.
  • Enter a file name and select Webpage, Complete (*.htm;*html) for the Save as type
  • Click the Save button.
In FireFox:
  • Navigate to the web page and select File, Save Page As... or right-click and select Save Page As...
  • Select or create a new folder to save the file, images, and associated items from the web page.
  • Enter a file name and select Web Page, complete (*.htm;*.html) for the Save as type
  • Click the Save button.
In Internet Explorer
  • Navigate to the web page and select File, Save Page As... or right-click and select Save Page As...
  • Enter a file name and select Web Page, complete (*.htm;*.html) for the Save as type
  • Click the Save button.
